# LLM-powered-chatbot
Terminal-based chatbot using **Google Gemini (Generative AI)** with secure API key handling


## Features

- Interactive chat through the terminal
- Uses 'google.generativeai' Python Client
- Gemini 2.0 Flash Model
- Secure API key handling vis environemnt Variable
- Minimal setup and easy to run
